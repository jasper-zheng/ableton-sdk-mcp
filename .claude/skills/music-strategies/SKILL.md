# Music-making strategies

Genre-neutral creative patterns. Use these to decide *what* to make; use the
**`ableton-midi`** skill (and other `ableton-*` skills) to *execute* it in Live.

## When to reach for this
- Staring at an empty project / no idea where to start.
- Have a loop but it feels static, or you're out of variation ideas.
- "Almost done" but can't arrange or end it.

## Finalized patterns
| File | Covers |
|------|--------|
| `strategies/melody.md` | Contour, motives, counterpoint, hocket, sound-color, lyric rhythm |
| `strategies/harmony.md` | Keys/scales, diatonic triads & function, 7ths/extensions, voice leading, parallel harmony, harmonic rhythm, repetition |
| `strategies/rhythm.md` | 3+3+2, beat programming (looseness, linear, ghost notes, top/bottom), bass+kick composite, polyrhythm, implied rhythm |
